# Chitkara Full Stack Engineering Challenge

## Tech Stack

* Backend: Node.js, Express.js
* Frontend: React.js, Vite
* Deployment: Render, Vercel

## Features

* Input validation
* Hierarchy tree generation
* Cycle detection
* Duplicate edge detection
* Depth calculation
* Summary generation

## API Endpoint

```http
POST /bfhl
```

### Sample Request

```json
{
  "data": ["A->B", "A->C", "B->D"]
}
```

## Run Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Deployment

* Backend URL: YOUR_RENDER_URL/bfhl
* Frontend URL: YOUR_VERCEL_URL
